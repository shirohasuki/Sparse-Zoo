## Sparse Zoo

### How to contribute

1.fork本仓库的add_workload分支，在本地新建一个文件夹添加新的workload

2.添加readme，说明怎么使用代码（如,如何提取矩阵乘）

3.我们需要的就是提取出稀疏计算的矩阵a和矩阵b（简单来说就是输出torch.matmul(a,b）的a和b

4.你可以添加个licence加上你的名字或者网名啥的，我们最后会一起合并到实验室仓库。

5.提交pr :blush:
